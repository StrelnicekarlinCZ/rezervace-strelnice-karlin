# V1.28a – finální oprava loga v e-mailu

- Logo v hlavičce potvrzovacího e-mailu má přesně 3 oblouky.
- Horní čára je vycentrovaná nad oblouky podle schváleného vzoru.
- Logo je vložené jako čisté inline HTML/CSS kvůli kompatibilitě e-mailových klientů.
- Zůstává produkční příprava V1.28: deploy, HTTPS, PWA a PostgreSQL postup.
