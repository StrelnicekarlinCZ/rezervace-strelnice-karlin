# V1.13 – překlady služeb + topbar + CTA

- Horní menu nahrazeno ikonou zámku pro budoucí skrytý vstup do adminu.
- Tlačítko Rezervovat termín má výraznější zelené podsvícení.
- Služby a podslužby vytvořené v adminu se v klientské části překládají i mimo pevně uložené ID.
- Doplněn slovník pro běžné české názvy a popisy služeb.
- U výběru času zůstává čas přímo v interaktivním tlačítku, duplicitní potvrzovací okno je odstraněné.

Poznámka: automatický překlad nově ručně vytvořených textů je v této lokální verzi řešen interním slovníkem a pravidly. Pro ostrou verzi doporučuji napojit překladový API modul nebo ukládat ke službě jazykové varianty.
