# SKIS v2.5.1 – Live Range Management

## Nové
- Ruční obsazení střeleckého stavu bez rezervace.
- Uvolnění ručně obsazeného stavu.
- Označení stavu jako mimo provoz.
- Vrácení stavu do provozu.
- Poznámka ke každému stavu.
- Přiřazení instruktora ke každému stavu.
- Reset ručního nastavení všech stavů.
- Lokální perzistence nastavení stavů přes localStorage.

## Zachováno
- Live Range výpočty obsazenosti.
- Volný stav = zelená.
- Obsazený stav = červená.
- VIP = modrá.
- Připravený stav = žlutá.
- Otevření klienta, odbavení, no-show, storno.

## Test
- Otevřít Live Range.
- Ručně obsadit volný stav.
- Ručně uvolnit stav.
- Označit stav mimo provoz.
- Vrátit stav do provozu.
- Vyplnit instruktora a poznámku.
- Refresh stránky – ruční nastavení má zůstat.
- Reset stavů – ruční nastavení se vyčistí.
