# V1.10 – časový roller, sdílení a jazyky

## Upraveno
- Krok 3 rezervace má výrazný blok „Vyberte čas“ s ikonou hodin.
- Časový roller se používá až po výběru dne a zobrazuje jen volné časy.
- Rezervace mezi službami sdílí stejný kalendář, takže se kontrolují překryvy napříč všemi službami.
- Karta rezervace má kopírovatelný odkaz a jen dvě rychlé ikony: e-mail a WhatsApp.
- Odebrány samostatné odkazy Signal/Facebook/Instagram.
- Přepínač jazyků mění hlavní klientské UI CZ / EN / DE / ES / FR.

## Poznámka
- Texty zadané ručně v adminu se zobrazují tak, jak jsou uložené. Pro ostrou vícejazyčnou verzi bude potřeba přidat jazykové varianty admin textů.
