# V1.6 – Kalendář a datový základ

Přidáno:
- admin kalendář na 7 dní dopředu,
- blokace časů pro konkrétní datum,
- export rezervací do CSV,
- export celého lokálního nastavení do JSON,
- příprava na ostré napojení PostgreSQL/Prisma,
- upravený text adminu na V1.6.

Poznámka: tato verze stále používá localStorage v prohlížeči, aby se rychle testovalo UI. Ostrá verze bude ukládat data na serveru do PostgreSQL, aby rezervace viděla všechna zařízení stejně.
