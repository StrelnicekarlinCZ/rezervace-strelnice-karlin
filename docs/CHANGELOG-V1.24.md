# CHANGELOG V1.24 – Mobile UI + PWA Splash/Icon Polish

## Hlavní zaměření

Verze V1.24 navazuje na V1.23 a řeší první produkční polish fázi:

1. doladění mobilního UI,
2. přípravu splash screenu a ikon aplikace,
3. lepší PWA metadata pro instalaci na mobil,
4. základní service worker cache pro instalovanou aplikaci.

## Mobilní UI

- Kompaktnější horní lišta.
- Lepší chování na malých displejích.
- Zmenšená hero sekce pro mobilní použití.
- Jemně zmenšené karty služeb a informační boxy.
- Lepší safe-area padding pro iPhone / moderní mobilní prohlížeče.
- Vylepšené spodní navigační menu.
- Lepší čitelnost detailu rezervace a QR karty na úzkém displeji.

## PWA

- Nový manifest s názvem COMBAT POWER – Rezervace střelnice.
- Přidány běžné ikony, Apple ikona a maskable ikony.
- Přidány splash screen obrázky pro iPhone/iPad.
- Doplněna Apple Web App metadata.
- Doplněn viewport fit pro safe-area displeje.
- Přidána registrace service workeru z klientské části.

## Poznámka k email notifikacím

Email notifikace pro klienta zůstává jako pevný další bod roadmapy po UI/PWA polish fázi.
